# dotfiles (.tilde)

personal configs and scripts for mac, zsh, vim, and git. tailored for working with kubernetes and docker.

## Install

clone this into `~/dev/projects/.tilde` and run the installer to symlink everything up.

```bash
cd ~/dev/projects/.tilde
./install.sh
```
(It safely backs up your existing dotfiles first so it won't destroy anything).

## Highlights
- **prompt:** simple day/light icon and git status
- **git:** some git aliases
- **docker/k8s:** image management and namespace switching mostly
- **macOS:** `Brewfile` to bootstrap a new laptop fast.
